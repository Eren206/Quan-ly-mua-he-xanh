Cách cài đặt chương trình dạng Source Code:
1. Chạy file SQL_MUAHEXANH.sql trên SQL Server
2. Chạy file SQL_master.sql trên SQL Server
3. Dùng Visual Studio mở source code lên và Chạy
4. Hoàn tất.
Cách cài đặt chương trình dạng PUBLISH:
1. Chạy file SQL_MUAHEXANH.sql trên SQL Server
2. Chạy file SQL_master.sql trên SQL Server
3. CHạy file setup.exe
4. Hoàn tất.
LƯU Ý: VÀO LẦN CHẠY ĐẦU TIÊN CHƯƠNG TRÌNH SẼ YÊU CẦU NHẬP TÊN SQL SERVER
